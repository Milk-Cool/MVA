# MVA
Milk Voice Assistant

I'm simple voice assistant written on JavaScript.

____

### You can say:

* Hello!
    * Hi!
* What's your name?
    * Who are you?
* How are you?
    * How are you today?
* What time is it?
    * Time?
* I'm good!
    * I'm great!
    * I'm wonderful!
* I'm bad.
    * I'm not so good.
    * I'm very bad.
* Stop.
    * Bye!
    * Goodbye!
    
____

___If you say something else, I'll search it in Google.___
